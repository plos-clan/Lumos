## 我认为

**一个编程语言应该是：可控的**

```lumos
while (true) { /* do nothing */ }
// 很明显这不应该是未定义行为
// 在 Lumos 中，它会产生一个死循环
```

```lumos
(i32)0x80000000 >> 1;
// 很明显它的值不应该是 0x40000000 0xc0000000 二选一
// 在 Lumos 中，它为 0xc0000000
```

**一个编程语言应该是：逻辑清晰的**

```lumos
list() + map();
// 很明显它们不应该能够相加
```
