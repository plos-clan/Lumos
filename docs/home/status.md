# 当前状态 {#title}

Lumos 目前以**语言设计与文档规范**为主，编译器实现处于探索阶段。

## 目前已有 {#current}

- 词法、语法与权限系统的文档草案。
- 标准库的设计纲要与接口规划。
- AST 与 IR 的结构化描述。

## 仍在进行 {#in-progress}

- 语法细节收敛与一致性审校。
- 编译器架构与实现路线评估。
- 标准库的具体语义与错误模型定义。

## 注意事项 {#notes}

- 文档可能频繁调整，语法与示例以最新规范为准。
- 旧版试验性源码不代表当前设计方向。
