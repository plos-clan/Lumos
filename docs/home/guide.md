# 阅读路线 {#title}

如果你是第一次接触 Lumos，建议按以下顺序阅读：

1. [概览](../overview.md)：快速理解语言定位与核心理念。
2. [语言参考](../language.md)：获取整体结构与章节入口。
3. [词法](../lexical/index.md)：明确词法规则与字面量规范。
4. [语法](../grammar/index.md)：掌握语法主体与关键字体系。
5. [权限系统](../grammar/permission.md)：理解副作用权限模型。
6. [标准库](../std/index.md)：了解标准库整体设计与命名规范。

如果你更关注编译器实现：

- [AST](../ast/index.md)
- [IR](../ir/index.md)
