# 阅读路线

如果你是第一次接触 Lumos，建议按以下顺序阅读：

1. `docs/overview.md`：快速理解语言定位与核心理念。
2. `docs/language.md`：获取整体结构与章节入口。
3. `docs/lexical/index.md`：明确词法规则与字面量规范。
4. `docs/grammar/index.md`：掌握语法主体与关键字体系。
5. `docs/grammar/permission.md`：理解副作用权限模型。
6. `docs/std/index.md`：了解标准库整体设计与命名规范。

如果你更关注编译器实现：

- `docs/ast/index.md`
- `docs/ir/index.md`

---

相关内容：语言入口见 [语言参考](../language.md)。
