
# 关键字

## 1

| 关键字 | 含义                       | 用法                   |
|--------|----------------------------|------------------------|
| var    | variable                   | 声明或定义变量         |
| val    | immutable variable         | 声明或定义不可变变量   |
| let    | expression                 | 定义表达式             |
| lit    | constant expression        | 定义常量表达式         |
| obj    | dynamically typed variable | 声明或定义动态类型变量 |
