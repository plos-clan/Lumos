
# 注释

Lumos 支持单行注释和多行注释，使用方式与 C 语言类似

## 单行注释

单行注释以 `//` 开头，直到行尾结束  
<span style="color:purple">注意 `//` 后面必须有一个空格，无空格会导致编译错误</span>

> token 中的 text 为注释内容，`//` 及之后的单字符空格不会被包含在 text 中

## 多行注释

多行注释以 `/*` 开头，以 `*/` 结尾  
<span style="color:purple">注意 `/*` 后和 `*/` 前必须有一个空格，无空格会导致编译错误</span>

> token 中的 text 为注释内容，`/*` 及之后的空白、`*/` 及之前的空白不会被包含在 text 中
>
> > 空白包括空格、制表符、回车等

<span style="color:green">不能嵌套的，别想了</span>

## 文档注释

文档注释用于为代码添加文档说明，支持单行和多行两种形式：

- 单行文档注释以 `///` 开头，直到行尾结束
- 多行文档注释以 `/**` 开头，以 `**/` 结尾

<span style="color:purple">注意 `///` 和 `/**` 后 `**/` 前必须有一个空格，无空格会导致编译错误</span>

文档注释必须紧挨着其所描述的代码元素（变量、函数、类等）之前，否则会导致编译错误。

```lumos
/// 这是一个示例函数
///
/// 它没有任何参数，返回类型为 `unit`
fun example_func() -> unit {
    // 函数体
}
```

```lumos
/**
 * 这是一个示例函数
 *
 * 它没有任何参数，返回类型为 `unit`
 **/
fun example_func() -> unit {
    // 函数体
}
```
